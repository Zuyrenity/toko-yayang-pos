# Sistem Informasi Penjualan dan Manajemen Stok (Point of Sales) - Toko Yayang

Proyek ini merupakan aplikasi Kasir (POS) berbasis web yang dikembangkan sebagai bagian dari proyek tugas akhir/skripsi. Sistem ini dirancang untuk mengoptimalkan proses transaksi penjualan, pengelolaan stok barang masuk/keluar, serta penataan laporan manajerial pada Toko Yayang.

## 🚀 Fitur Utama
* **Autentikasi Multi-User:** Memiliki 2 hak akses berbeda, yaitu **Admin** (untuk operasional kasir & gudang) dan **Pimpinan** (untuk memantau laporan).
* **Manajemen Barang (Stok):** Fitur CRUD data produk beserta pembaruan jumlah stok secara otomatis.
* **Pencatatan Barang Masuk:** Mengelola log suplai barang yang masuk ke toko.
* **Transaksi Kasir (Real-time):** Proses input penjualan dengan kalkulasi otomatis untuk total harga, nominal bayar, dan kembalian.
* **Laporan Penjualan:** Menyimpan riwayat transaksi mendetail untuk kebutuhan analisis pimpinan.

## 🛠️ Teknologi yang Digunakan
* **Back-End:** PHP
* **Front-End:** HTML5, CSS3, JavaScript, Bootstrap, FontAwesome
* **Database:** MySQL / MariaDB
